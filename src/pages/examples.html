Pie Chart Example

Pie chart with custom color array and data:

Usage

Static (HTML attributes):

<ui-pie-chart
  title="Stock by Category"
  data='[{"label":"Apples","value":30,"color":"#e57373"}, {"label":"Bananas","value":20,"color":"#ffd54f"}]'
  colors='["#e57373","#ffd54f"]'
  height="300"
  width="100%"
  legend-position="right"
></ui-pie-chart>

Dynamic (JS properties):

import { getPieChartProducts } from "../api/example-pie-chart-data";
getPieChartProducts().then(res => {
  const data = Object.values(res.products.reduce((acc, p) => {
    if (!acc[p.category]) acc[p.category] = { label: p.category, value: 0 };
    acc[p.category].value += p.stock;
    return acc;
  }, {}));
  const pie = document.getElementById("pie-demo");
  if (pie) pie.setAttribute("data", JSON.stringify(data));
});

Features

  • Legend displays color and label for each data set
  • Legend can be positioned: top, bottom, left, right (default: bottom)
  • Click or press Enter/Space on a legend item to toggle that set
  • Hover or focus a legend item to highlight the corresponding chart section

Attributes

  • data: JSON array of { label, value, color? } objects
  • colors: JSON array of fallback colors (optional)
  • title: Optional string — displays a bold title above the chart
  • height: Number (chart height in px, default 200)
  • width: CSS width value (default 100%)
  • legend-position: top | bottom | left | right (default: bottom)

Bar Chart Example

Bar chart with custom color array and data:

Usage

Static (HTML attributes):

<ui-bar-chart
  title="Avg Price by Category"
  data='[{"label":"Q1","value":40,"color":"#42a5f5"}, {"label":"Q2","value":55,"color":"#66bb6a"}]'
  colors='["#42a5f5","#66bb6a"]'
  height="300"
  width="100%"
  font-size="12"
  legend-position="right"
></ui-bar-chart>

Dynamic (JS properties):

import { getBarChartProducts } from "../api/example-bar-chart-data";
getBarChartProducts().then(res => {
  const totals = res.products.reduce((acc, p) => {
    if (!acc[p.category]) acc[p.category] = { sum: 0, count: 0 };
    acc[p.category].sum += p.price;
    acc[p.category].count++;
    return acc;
  }, {}));
  const data = Object.entries(totals).map(([label, { sum, count }]) => ({ label, value: Math.round(sum / count) }));
  const bar = document.getElementById("bar-demo");
  if (bar) bar.setAttribute("data", JSON.stringify(data));
});

Features

  • Legend displays color and label for each data set
  • Legend can be positioned: top, bottom, left, right (default: bottom)
  • Click or press Enter/Space on a legend item to toggle that set
  • Hover or focus a legend item to highlight the corresponding bar
  • Reference lines overlay dashed horizontal lines at target values across the chart

Attributes

  • data: JSON array of { label, value, color? } objects
  • colors: JSON array of fallback colors (optional)
  • title: Optional string — displays a bold title above the chart
  • height: Number (chart height in px, default 200)
  • width: CSS width value (default 100%)
  • legend-position: top | bottom | left | right (default: bottom)
  • reference-lines: JSON array of { value, label?, color? } — overlays dashed horizontal lines at the given values (vertical mode only)

Bar Chart with Reference Lines

Use reference-lines to overlay dashed horizontal target lines at specific values. Each line can have an optional label and color. Reference lines appear in the legend and can be toggled on/off. The Y-axis scale automatically expands to include them.

Usage

<ui-bar-chart
  data='[{"label":"Jan","value":42},{"label":"Feb","value":58}]'
  reference-lines='[{"value":60,"label":"Target","color":"#ef5350"}]'
></ui-bar-chart>

Reference line fields

  • value: Number — the Y-axis position of the line
  • label: Optional string — shown inline on the chart and in the legend
  • color: Optional CSS color — defaults to #ef5350

Bar Chart with Line Series Overlay

Use line-series to overlay one or more data series as connected lines with dot markers on top of the columns. Each series has a values array aligned to the bar categories. Series appear in the legend and can be toggled on/off independently.

Usage

<ui-bar-chart
  data='[{"label":"Jan","value":42},{"label":"Feb","value":58}]'
  line-series='[{"label":"Last Year","values":[35,50],"color":"#ff7043"}]'
></ui-bar-chart>

Line series fields

  • label: Series name — shown in the legend and tooltip
  • values: Array of numbers, one per bar category (must match bar count)
  • color: Optional CSS color — auto-assigned if omitted

Horizontal Bar Chart Example

Bar chart displayed as horizontal bars — useful when labels are long or there are many categories:

Usage

Static (HTML attributes):

<ui-bar-chart
  title="Avg Price by Category"
  horizontal
  data='[{"label":"Q1","value":40,"color":"#42a5f5"}, {"label":"Q2","value":55,"color":"#66bb6a"}]'
  colors='["#42a5f5","#66bb6a"]'
  width="100%"
  font-size="12"
  legend-position="left"
></ui-bar-chart>

Attributes

  • horizontal: Boolean — renders bars horizontally instead of vertically
  • title: Optional string — displays a bold title above the chart
  • font-size: Number in px for label text size (default 12); axis labels scale proportionally
  • All other ui-bar-chart attributes apply (data, colors, width, legend-position)
  • The height attribute is not used in horizontal mode — chart height auto-sizes to fit all bars

Line Chart Example

Multi-series line chart with interactive tooltips on each data point:

Usage

Static (HTML attributes):

<ui-line-chart
  title="My Chart"
  x-labels='["Jan","Feb","Mar"]'
  data='[{"label":"Series A","values":[10,25,18]},{"label":"Series B","values":[5,12,20]}]'
  colors='["#42a5f5","#66bb6a"]'
  height="400"
  width="100%"
  legend-position="bottom"
></ui-line-chart>

Dynamic (JS properties):

import { getLineChartProducts, buildLineChartData } from "../api/example-line-chart-data";

getLineChartProducts().then(res => {
  const { categories, series } = buildLineChartData(res);
  const chart = document.getElementById("line-demo");
  chart.setAttribute("x-labels", JSON.stringify(categories));
  chart.setAttribute("data", JSON.stringify(series));
});

Features

  • Multiple series displayed as separate labelled lines
  • Tooltips on every data point showing series name, x-label, and value
  • Legend can be positioned: top, bottom, left, right (default: bottom)
  • Click or press Enter/Space on a legend item to toggle that series
  • Hover a legend item to highlight its line and dim others
  • Keyboard accessible: Tab to each data point, tooltip shown on focus

Attributes

  • data: JSON array of { label, values: number[], color? } series objects
  • x-labels: JSON array of strings for the x-axis category labels
  • colors: JSON array of fallback colors (used when a series has no color)
  • title: Optional string — displays a bold title above the chart
  • height: Number (chart height in px, default 300)
  • width: CSS width value (default 100%)
  • legend-position: top | bottom | left | right (default: bottom)

Attributes

  • new-window: Opens in a new tab with target="_blank" rel="noopener noreferrer" — router is bypassed
  • button: Boolean — renders as a button-styled anchor
  • variant: success | warning | alert — applies variant colour (button mode only)
  • disabled: Boolean — disables navigation and applies disabled styles (button mode only)

Search

Add the search attribute to render a live search input above the table. Typing filters rows across all columns (case-insensitive). Combine with csv-export and zebra as needed.

Usage

Static (HTML attributes):

<ui-table
  search
  csv-export
  csv-filename="report"
  zebra
  columns='[{"key":"name","label":"Name","sortable":true}]'
  data='[{"name":"Alice"},{"name":"Bob"}]'
></ui-table>

Dynamic (JS properties):

const table = document.querySelector('ui-table');
table.columns = [{ key: 'name', label: 'Name', sortable: true }];
table.data = [{ name: 'Alice' }, { name: 'Bob' }];

Attributes

  • columns: JSON array of { key, label, sortable? } objects
  • data: JSON array of row objects keyed to column keys
  • search: Boolean — shows a live search input that filters rows across all columns
  • csv-export: Boolean — shows the Export CSV button above the table
  • csv-filename: Download filename without extension (default: "export")
  • zebra: Boolean — applies alternating row background colours

Sorting

  • Click a sortable column header to cycle: ascending → descending → unsorted
  • Keyboard accessible — Enter or Space activates sort on focused headers
