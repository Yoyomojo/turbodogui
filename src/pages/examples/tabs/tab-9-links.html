Button Links

Add the button attribute to render the link styled identically to <td-button>. Supports the same variant and disabled attributes. The element still navigates via the router.

Link Sizes

The size attribute controls the padding and font size of button-styled links. Available sizes: small, medium (default), large, extra-large. Only applies when the button attribute is present.

Attributes

  • href / to: Route path or URL
  • label: Link text (falls back to element text content)
  • new-window: Opens in a new tab with target="_blank" rel="noopener noreferrer": router is bypassed
  • button: Boolean: renders as a button-styled anchor
  • size: small | medium | large | extra-large: controls padding and font size (button mode only, default medium)
  • variant: success | warning | alert: applies variant colour (button mode only)
  • disabled: Boolean: disables navigation and applies disabled styles (button mode only)
