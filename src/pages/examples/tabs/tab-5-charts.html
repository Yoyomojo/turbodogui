Bar Chart with Reference Lines

Use reference-lines to overlay dashed horizontal target lines at specific values. Each line can have an optional label and color. Reference lines appear in the legend and can be toggled on/off. The Y-axis scale automatically expands to include them.

Usage

Reference line fields

  • value: Number: the Y-axis position of the line
  • label: Optional string: shown inline on the chart and in the legend
  • color: Optional CSS color: defaults to #ef5350

Bar Chart with Line Series Overlay

Use line-series to overlay one or more data series as connected lines with dot markers on top of the columns. Each series has a values array aligned to the bar categories. Series appear in the legend and can be toggled on/off independently.

Usage

Line series fields

  • label: Series name: shown in the legend and tooltip
  • values: Array of numbers, one per bar category (must match bar count)
  • color: Optional CSS color: auto-assigned if omitted

Grouped Bar Chart Example

Multiple series per category: each group of bars shares the same X-axis label:

Usage

bar-series fields

  • label: Series name shown in the legend and tooltip
  • values: Array of numbers: one value per category
  • color: Optional CSS color: auto-assigned from colors or HSL if omitted

Attributes

  • bar-series: JSON array of { label, values, color? }: activates grouped mode
  • categories: JSON string array: X-axis category labels (defaults to "1", "2", â€¦ if omitted)
  • All other td-bar-chart attributes apply (title, height, width, legend-position, horizontal, font-size)
  • Click or press Enter/Space on a legend item to toggle that series
  • Works in horizontal mode too: each row becomes a group of horizontal bars
  • group-tooltip: Boolean — when present, hovering any bar highlights all bars in that category group and shows a combined tooltip listing every series value for that group

Bubble Chart Example

Three-dimensional data: x-axis, y-axis, and bubble size. Supports multiple datasets with interactive legend, hover tooltips, and optional value labels inside each bubble.

Usage

Static (HTML attribute):

Dynamic (API):

JS property setter:

Features

  • Multiple series with per-series colors and toggleable legend
  • Bubble size (r) scaled proportionally; smallest bubble is always visible
  • X and Y axes auto-scale with nice tick intervals
  • Hover/focus any bubble to see a tooltip with series name, point label, x, y, and size
  • Hover a legend item to highlight that series; click to toggle it on/off
  • Load data via data attribute (JSON), JS .data setter, or src attribute (API fetch)
  • Optional axis name labels via x-label / y-label

Attributes

  • data: JSON array of series objects - [{ label, color?, points: [{x, y, r, label?}] }]
  • src: URL to fetch JSON data from; expects same shape as data
  • colors: JSON array of fallback colors applied to series in order
  • title: Bold title above the chart
  • x-label: Axis label along the x-axis
  • y-label: Axis label along the y-axis (rotated 90°)
  • height: Chart height in px (default 360)
  • width: CSS width value (default 100%)
  • legend-position: top | bottom | left | right (default: bottom)
  • show-values: Boolean: renders the r value inside each bubble
