Attributes

  • type: HTML input type (text, email, password, tel, number, url, search, etc.)
  • label: Display label for the input
  • placeholder: Placeholder text
  • size: small | medium | large | extra-large (default: medium)
  • disabled: Boolean attribute to disable the input
  • error: Boolean attribute to show error state
  • show-password-toggle: Boolean attribute (password type only) — adds an eye icon to reveal/hide the password
