Breadcrumb Component

The breadcrumb component provides accessible page-trail navigation. It supports three usage modes: plain HTML markup inside the tags, a JSON items attribute, and dynamic population from an API via src or the JavaScript items setter.

HTML Slot Mode

Place <li> elements directly inside <td-breadcrumb>. The component automatically injects separators between items using the separator attribute (default /). Mark the active crumb with aria-current="page":

    Attributes & API

    Attributes

    • items: JSON array of { label: string, href?: string } objects
    • src: URL to fetch the items array from
    • separator: Divider string between crumbs (default /)

    JavaScript

    • items - get/set BreadcrumbItem[]; triggers a re-render

    Events

    • breadcrumb-click - fires when a non-current crumb is clicked; detail: { index, label, href }

    Slot (HTML) mode

    • Supply <li> children - the component clones them into a shadow <ul> and injects separator items automatically
    • Mark the last crumb with aria-current="page" to suppress its link

    Accessibility

    • Wrapped in <nav aria-label="Breadcrumb">
    • Both modes render a semantic <ul> with <li> items
    • Last item carries aria-current="page"
