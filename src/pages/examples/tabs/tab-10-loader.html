Component Features

  • Message: Optional loading text display
  • Size: Customize spinner size (default: 48px)
  • Color: Customize spinner color (default: primary)
  • Variant: "spinner" or "dots" animations
  • Contained: Boolean attribute for positioning within a container instead of full-screen
  • Overlay: Full-screen overlay with fade animation (or contained)
  • Methods: show() and hide()
  • Events: Emits loader-show and loader-hide
