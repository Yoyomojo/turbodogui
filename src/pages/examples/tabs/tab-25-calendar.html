Calendar Component

An inline calendar for selecting individual dates or continuous date ranges. Fully keyboard-accessible with hover preview for range selections.

Use mode="single" (default) for individual date selection or mode="range" for start/end date ranges. Constrain selectable dates with min and max (ISO 8601 strings).

Single Date Selection

Fires a change event with detail.value (ISO date string) when a date is clicked.

No date selected

Pre-selected Date

Pass value to start with a date already selected.

Date Range Selection

Set mode="range". Click once for the start, again for the end. Hover to preview the range. Fires change with detail.start and detail.end.

No range selected

Pre-selected Range

Set both range-start and range-end to initialise with an existing range.

Min / Max Constraints

Use min and max to restrict which dates can be selected. Dates outside the range appear dimmed and are not interactive.

Programmatic API

Use setValue(), setRange(), clear() and getValue() / getRangeStart() / getRangeEnd() directly on the element.

Usage

<td-calendar value="2026-07-23"></td-calendar> <td-calendar mode="range" range-start="2026-07-10" range-end="2026-07-20"> </td-calendar> <td-calendar min="2026-07-01" max="2026-07-31"></td-calendar> // Single mode calendar.addEventListener('change', (e) => { console.log(e.detail.value); // "2026-07-23" }); // Range mode — partial (start selected, awaiting end) calendar.addEventListener('range-change', (e) => { console.log(e.detail.start, e.detail.end); // "2026-07-10", null }); // Range mode — complete calendar.addEventListener('change', (e) => { console.log(e.detail.start, e.detail.end); // "2026-07-10", "2026-07-20" }); const cal = document.querySelector('td-calendar'); // Single cal.setValue('2026-07-23'); cal.getValue(); // "2026-07-23" // Range cal.setRange('2026-07-10', '2026-07-20'); cal.getRangeStart(); // "2026-07-10" cal.getRangeEnd(); // "2026-07-20" cal.clear(); // clears value, range-start, range-end

Attributes

Attribute Type Description
mode "single" | "range" Selection mode. Defaults to "single".
value string Selected date in YYYY-MM-DD format (single mode).
range-start string Range start date in YYYY-MM-DD format.
range-end string Range end date in YYYY-MM-DD format.
min string Earliest selectable date (YYYY-MM-DD).
max string Latest selectable date (YYYY-MM-DD).
