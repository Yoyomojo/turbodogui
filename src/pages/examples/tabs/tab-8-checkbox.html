Checkbox Component

A styled checkbox with label, states, and accessible error handling.

States

Checkbox Usage

Attributes

  • label: Text label displayed beside the checkbox
  • name: Form field name
  • value: Submitted value when checked (default: on)
  • checked: Boolean - pre-checked state
  • disabled: Boolean - disables interaction
  • required: Boolean - marks field as required
  • error: Boolean - shows error state and message

Events

change - fires on toggle: detail: { checked: boolean, value: string }
