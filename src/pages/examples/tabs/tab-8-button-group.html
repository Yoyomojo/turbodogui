Usage

Use a button group for related actions in a single context. Keep button order intentional, and use pill when you want a fully rounded segmented control.

Attributes

  • wrap: Boolean: allows horizontal groups to wrap onto multiple lines
  • gap: CSS gap value between items (default: 0.5rem)
  • align: start | center | end | stretch
  • justify: start | center | end | between | around | evenly
  • full-width: Boolean: makes the group use full available width
  • pill: Boolean: fully rounds the left side of the first button and the right side of the last button
