Multi-Select Component

A checkbox-based multi-select list. Add the search attribute to show a live search field above the options.

Basic Multi-Select

A scrollable list of checkbox options. Selected values are returned as an array via the change event.

HTML-Populated Multi-Select

Define options directly with HTML <option> and <optgroup> children.

Multi-Select with Search

Add the search attribute to render a live search input above the options list.

API-Populated Multi-Select

Options loaded dynamically from an API.

Multi-Select Sizes

Multi-select supports the same size options as text inputs and buttons.

Static (HTML attributes):

Dynamic (JS properties):

Attributes

  • label: Display label above the list
  • size: small | medium | large | extra-large (default: medium)
  • options: JSON array of { value, label } objects (optional if HTML <option> children are provided)
  • value: JSON array of initially selected values
  • search: Boolean - shows a search input above the options
  • search-placeholder: Placeholder text for the search input (default: "Search...")
  • select-all: Boolean - adds a Select all / Deselect all row at the top of the list
  • placeholder: Trigger placeholder text when nothing is selected. Falls back to label if not set.
  • Options: Supports HTML <option> and <optgroup> children as an alternative to the options JSON attribute.

Events

  • change: Fires on every toggle. detail.value is the full array of currently selected values.

JS properties

  • options: get/set the options array
  • value: get/set the selected values array
