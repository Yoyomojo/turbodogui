Mosaic

<td-mosaic> is a CSS Grid-based mosaic layout (Instagram-style). Rows auto-size to keep cells square. Use <td-mosaic-item> for each cell, with col-span and row-span to create featured tiles. Hover over a cell to see the scale and overlay effect.

Attributes

<td-mosaic>

Attribute Default Description
columns 3 Number of equal-width columns
gap 4px CSS gap between cells
row-height auto (square) Override row height. Omit for auto-square cells.

<td-mosaic-item>

Attribute Default Description
col-span 1 Number of columns to span
row-span 1 Number of rows to span
slot="overlay" - Content shown on hover overlay
