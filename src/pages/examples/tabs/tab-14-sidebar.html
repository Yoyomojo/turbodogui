Features

  • Collapse / expand via toggle arrow button
  • Collapsed state shows icons with tooltip labels on hover/focus
  • Optional search attribute adds a live filter input
  • Section labels can be stacked to create nested groups (for example Components and Inputs)
  • Items accept any HTML icon (SVG, emoji, text character)
  • Optional badge on any item (count, label)
  • Active item highlighted via active attribute or activeKey setter
  • Mobile: full-width drawer with overlay, opened via sidebar.open()
  • Keyboard: ArrowUp / ArrowDown navigate items, Home / End jump to first/last
  • Slots: header, footer, body for arbitrary markup
  • Fires nav-item-click and sidebar-toggle events

Attributes

Attribute Description
items JSON array of SidebarItem objects
active Key of the initially active item
collapsed Boolean: start in collapsed state
search Boolean: show the search / filter input
search-placeholder Placeholder text for the search input
label Accessible aria-label for the nav element
no-toggle Boolean: hide the collapse/expand toggle button

SidebarItem fields

Field Description
key Unique string identifier
label Display text and collapsed tooltip
icon HTML string: SVG, emoji, or any markup
href If set, renders an <a> tag instead of a button
badge Optional pill badge text
section true: renders as a non-interactive section label
data Arbitrary payload passed through in the click event
