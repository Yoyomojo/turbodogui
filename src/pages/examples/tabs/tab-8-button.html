Button Usage

Full Width

Add the full-width boolean attribute to make the button fill its parent container:

Full Width Button

Attributes

  • type: HTML button type (button, submit, reset, default: button)
  • variant: Visual style (success, warning, alert, or default/primary)
  • size: small | medium | large | extra-large (default: medium)
  • disabled: Boolean attribute to disable the button
  • full-width: Boolean attribute to make the button fill 100% of its parent
  • label: Alternative way to set button text (uses textContent if not provided)
