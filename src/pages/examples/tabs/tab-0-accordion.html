Component Features

  • Items: Number of accordion sections
  • Item Titles: Use item-N-title attribute for each section title
  • Allow Multiple: Boolean attribute to allow multiple sections open simultaneously
  • Keyboard Navigation: Arrow keys to navigate, Home/End for first/last items
  • Accessibility: Full ARIA roles and labels for screen readers
  • Smooth Animations: CSS transitions for expand/collapse
  • Events: Emits accordion-toggle event with active index
