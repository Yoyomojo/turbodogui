Code Examples

Attributes

  • value — Current progress value (default: 0)
  • max — Maximum value (default: 100)
  • label — Accessible label shown above the bar
  • show-value — Displays the percentage alongside the label
  • variant — success | warning | danger
  • striped — Applies a diagonal stripe pattern
  • animated — Animates the stripes (use with striped)
  • indeterminate — Pulsing bar for unknown progress duration
  • height — CSS length for bar height (e.g. 4px, 1rem)
  • color — Custom CSS color for the bar fill

JavaScript API

  • value (get/set) — Current numeric value
  • max (get/set) — Maximum numeric value
  • percentage (get) — Computed 0–100 percentage

CSS Custom Properties

  • --progress-height — Bar height (overridden by height attribute)
  • --progress-color — Bar fill color (overridden by color attribute)
