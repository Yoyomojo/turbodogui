Dropdown Menu Component

A flexible dropdown menu that supports HTML slot items, JSON configuration, and remote API loading. Includes full keyboard navigation and ARIA support.

HTML Slot Items

Pass items directly as child HTML elements. Any <a> or <button> inside the component becomes a menu item.

JSON Items

Pass items as a JSON array via the items attribute. Supports labels, values, hrefs, dividers, and disabled items.

No item selected yet.
<td-dropdown label="File" items='[ {"label":"New File","value":"new"}, {"label":"Open…","value":"open"}, {"label":"Save","value":"save"}, {"label":"---"}, {"label":"Export as PDF","value":"export-pdf"}, {"label":"Delete","value":"delete","disabled":true} ]' ></td-dropdown>

Items with Icons

Add SVG or emoji icons to JSON items via the icon property. The value is rendered as raw HTML in the icon slot.

[ { "label": "Profile", "value": "profile", "icon": "<svg>…</svg>" }, { "label": "Settings", "value": "settings", "icon": "<svg>…</svg>" }, { "label": "---" }, { "label": "Sign Out", "value": "signout", "icon": "<svg>…</svg>" } ]

Remote API Loading

Set the src attribute to a URL that returns a DropdownItem[] JSON array. Items are fetched lazily on first open and cached for subsequent opens.

This demo fetches user names from JSONPlaceholder and pre-maps the response to the expected format using a small setup script below.

<!-- Your endpoint must return DropdownItem[] --> <td-dropdown label="Load from API" src="/api/menu-items.json"></td-dropdown> <!-- Expected JSON format: --> [ { "label": "Dashboard", "value": "dashboard", "href": "/dashboard" }, { "label": "Reports", "value": "reports" }, { "label": "---" }, { "label": "Settings", "value": "settings" } ] <!-- Or map a third-party API response in your own script: --> <script> fetch('https://jsonplaceholder.typicode.com/users') .then(r => r.json()) .then(users => { const items = users.map(u => ({ label: u.name, value: String(u.id) })); dropdown.setAttribute('items', JSON.stringify(items)); }); </script>

Placement

Control where the menu opens relative to the trigger using placement: bottom-start (default), bottom-end, top-start, or top-end.

Events
