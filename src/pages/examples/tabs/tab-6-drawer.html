Drawer Component

A slide-in panel that can appear from any edge of the viewport. Wraps content in a focusable overlay with an accessible close button. Use the position attribute to control which edge the drawer slides from. Left and right drawers are always full viewport height; top and bottom drawers span the full viewport width.

Position Variants

Click each button to open a drawer from that edge:

Right (default) Left Top Bottom

This drawer slides in from the right. Default width is 320px.

This drawer slides in from the left.

Custom Size

Use width for left/right drawers and height for top/bottom drawers:

Wide Right (500px) Tall Bottom (50vh)

This drawer is 500px wide.

This drawer takes up 50vh of the screen height.

Usage

Attributes

  • position: right (default) | left | top | bottom
  • width: Panel width for left/right drawers (default 320px)
  • height: Panel height for top/bottom drawers (default 320px)
  • title: Header title text
  • open: Boolean - present when the drawer is visible

JS API

  • drawer.open() - opens the drawer
  • drawer.close() - closes the drawer

Events

  • drawer-close - fires when the drawer is dismissed (close button, overlay click, or Escape)
