Carousel Component

A responsive, accessible slideshow. Left/right arrows only appear when adjacent slides exist. Dot indicators show the active slide. Supports images, rich HTML content, touch swipe, keyboard navigation, and optional autoplay. Works inline or inside a modal.

Attributes

  • label: ARIA region label for screen readers (default: "Carousel")
  • loop: Boolean: wrap around on first/last slide; both arrows always visible
  • autoplay: Boolean: automatically advance slides
  • interval: Number (ms): autoplay speed, default 4000. Minimum 500

Slots

  • slide-0, slide-1, slide-2 ... each child element with a matching slot attribute becomes a slide. Any HTML is accepted: images, text, cards, forms.

Events

  • slide-change fires after every slide transition. detail: { index: number, total: number }

JS Properties

  • currentIndex getter: current zero-based slide index
  • goTo(n) navigate to index n
  • next() / prev() advance or go back one slide

Keyboard

  • Arrow buttons are fully keyboard focusable
  • ArrowLeft / ArrowRight when carousel is focused navigate slides
  • Dot indicators are a tablist: use Tab to reach them
