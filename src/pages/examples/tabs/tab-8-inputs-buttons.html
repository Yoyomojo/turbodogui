File Input Usage

Attributes

  • label: Label shown above the drop zone
  • accept: Allowed file types using standard file input syntax (for example image/* or .pdf,.docx)
  • multiple: Boolean - allows selecting more than one file
  • disabled: Boolean - disables click and drag/drop selection

Defaults

  • When accept is not provided, it defaults to */* (all file types)

Events

change - emits detail.files and detail.names after file selection.

Multi-Select Usage

Static (HTML attributes):

Dynamic (JS properties):

Attributes

  • label: Display label above the list
  • options: JSON array of { value, label } objects
  • value: JSON array of initially selected values
  • search: Boolean - shows a search input above the options
  • search-placeholder: Placeholder text for the search input (default: "Search...")
  • select-all: Boolean - adds a Select all / Deselect all row at the top of the list
  • placeholder: Trigger placeholder text when nothing is selected. Falls back to label if not set.

Events

  • change: Fires on every toggle. detail.value is the full array of currently selected values.

JS properties

  • options: get/set the options array
  • value: get/set the selected values array

Attributes

  • variant: Visual style (success, warning, alert, or default/primary)
  • size: small | medium | large | extra-large (default: medium)
  • disabled: Boolean attribute to disable the button
  • full-width: Boolean attribute to make the button fill 100% of its parent
  • label: Alternative way to set button text (uses textContent if not provided)
