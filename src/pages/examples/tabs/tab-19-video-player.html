Video Player

<td-video-player> is a fully custom video player component with support for local files (drag-and-drop or browse), external URLs via the src attribute, playback controls, volume, and video sharing with a timestamped link.

Local file player

Add a file to the src attribute and load it locally. The "Share" button copies a timestamped link with the current playback time to the clipboard, which will work as long as the same file is loaded again.

External source

Set the src attribute to any publicly accessible video URL. The Share button copies a timestamped link (?t=<seconds>) to the clipboard.

Basic usage

Attributes

Attribute Description
src URL of the video to load
title Overlay title shown at the top of the video
poster Thumbnail image shown before playback starts
autoplay Start playing automatically (use with muted)
muted Start with audio muted
loop Loop playback continuously

Append ?t=<seconds> to any page URL and the player will automatically seek to that time when the video metadata loads.
