Component Features

  • Labels: Pipe-separated string of tab labels
  • Default Tab: Index of the tab to show initially (default: 0)
  • Keyboard Navigation: Arrow keys, Home, End for accessibility
  • Events: Emits tab-changed event with active index
  • Slots: Use slot="tab-N" for each tab's content
