Usage

Attributes

  • label: Text label above the upload area
  • size: small | medium | large | extra-large (default: medium)
  • accept: Allowed file types (image/*, .pdf,.docx, application/pdf). Default: */* (all types)
  • mode: single | multiple (optional explicit selection mode)
  • multiple: Boolean - allows selecting more than one file (default is single)
  • max-size: Maximum size per file. Supports bytes (for example 1048576) or units (500KB, 5MB, 1GB)
  • drop-title: Custom heading text inside the drop zone
  • drop-subtitle: Custom helper text under the heading
  • accepted-label: Prefix text for accepted file type display
  • max-size-label: Prefix text for max size display
  • no-files-label: Text shown when no file is selected
  • disabled: Boolean - disables click and drag/drop interaction

Slots

  • content: Replace the default drop-zone copy with custom markup via slot="content"

Events

  • change: Fires after selection with detail: { files: File[], names: string[] }
  • file-reject: Fires when files are filtered out by type or size

Select files to see event output.
