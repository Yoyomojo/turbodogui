Attributes

  • label: Display label for the textarea
  • placeholder: Placeholder text
  • value: Initial textarea value
  • name: Form field name
  • size: small | medium | large | extra-large (default: medium)
  • rows: Number of visible text lines (default: 4)
  • resize: CSS resize mode (default: vertical)
  • required: Boolean attribute to require input
  • disabled: Boolean attribute to disable the textarea
  • error: Boolean attribute to show error state
