Foundational components used throughout the library: <td-card> for surface containers, <td-code> for syntax-highlighted code blocks, and <td-theme-toggle> for switching between light and dark mode.

<td-theme-toggle>

Live demo

The button in the top-right of the app is a <td-theme-toggle>. You can also embed one anywhere:

Usage

Reads the current theme from localStorage via getTheme() and calls toggleTheme() on click. The host <html> element receives a data-theme="dark" attribute which drives all CSS custom-property values defined in src/theme/base-theme.css.
