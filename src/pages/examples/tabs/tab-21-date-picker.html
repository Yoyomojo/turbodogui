Date Picker Component

A fully accessible, keyboard-navigable date picker built as a custom element. Supports size variants, validation states, min/max constraints, and form submission via a hidden input.

Display Formats

Use the display-format attribute to control how the selected date appears in the input field. The underlying value and the change event always use YYYY-MM-DD. All pickers below share the value 2026-07-14.

Min / Max Constraints

Use min and max (ISO 8601 date strings) to restrict selectable dates. Disabled dates are shown faded and cannot be selected.

Usage

Attributes

  • label: Label text displayed above the picker
  • name: Name for form submission (hidden input)
  • value: Selected date in YYYY-MM-DD format
  • placeholder: Placeholder text when no date is selected
  • display-format: How the selected date is shown in the input field (default: MMMM D, YYYY). The stored value and change event are always YYYY-MM-DD.
  • size: small | medium | large | extra-large (default: medium)
  • min: Earliest selectable date (YYYY-MM-DD)
  • max: Latest selectable date (YYYY-MM-DD)
  • disabled: Disables the picker
  • required: Marks the field as required
  • error: Shows the error state with a red border

Keyboard Navigation

  • Enter / Space / ↓ on the input — opens the calendar
  • Arrow keys — move focus between days
  • Enter / Space on a day — selects it
  • Escape — closes the calendar without selecting
