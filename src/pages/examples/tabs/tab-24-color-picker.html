Attributes

  • value — Initial color; accepts #rrggbb, 8-digit hex, or rgb()/rgba()
  • label — Field label shown above the trigger
  • format — Output format: hex (default) | rgb | hsl
  • alpha — Enables the opacity slider and alpha channel in the output value
  • disabled — Prevents opening the picker
  • size — small | medium (default) | large | extra-large
  • presets — JSON array of hex strings rendered as clickable swatches

change Event Detail

  • value — Formatted string per the format attribute
  • hex — Always #rrggbb regardless of format
  • rgb — [r, g, b] tuple (0–255 each)
  • alpha — Number 0–1

Keyboard Navigation

  • Gradient area: Arrow keys move saturation (←/→) and brightness (↑/↓); hold Shift for 10× steps
  • Hue / Alpha sliders: Arrow keys adjust value; hold Shift for 10× steps
  • Escape: Closes the panel and returns focus to the trigger
